# 📑 Batch PDF Merger (Конвейер объединения PDF)

Простой и эффективный инструмент на Python с графическим интерфейсом для пакетного объединения PDF-документов. В отличие от обычных объединителей, этот скрипт работает по принципу «конвейера»: вы заранее планируете, сколько итоговых файлов вам нужно, и пошагово наполняете каждый из них.

## ✨ Особенности
*   **Интерфейс на CustomTkinter**: Современный дизайн в темных/светлых тонах.
*   **Логика «Мастера» (Wizard)**: Пошаговая настройка каждого выходного файла.
*   **Пакетная обработка**: Создание нескольких разных PDF-сборок за один рабочий цикл.
*   **Предварительный просмотр**: Список выбранных файлов отображается перед переходом к следующему этапу.
*   **Финальное подтверждение**: Сводка всех запланированных задач перед записью на диск.

## 📖 Как пользоваться
1.  **Настройка циклов**: Введите количество итоговых файлов, которые вы хотите получить на выходе.
2.  **Выбор файлов**: Для каждого «цикла» выберите один или несколько PDF-документов. Программа запомнит их порядок.
3.  **Подтверждение**: После настройки всех циклов проверьте сводку в появившемся окне.
4.  **Сохранение**: Выберите папку, в которую программа сохранит все готовые файлы (имена будут присвоены автоматически: `result_batch_1.pdf` и т.д.).
